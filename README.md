# 🚀 Accredian Enterprise Clone

## 📌 Project Overview

This project is a partial clone of the Accredian Enterprise website built using Next.js.
It focuses on creating a clean, responsive, and structured landing page with reusable components.

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React (Functional Components & Hooks)
* Tailwind CSS

---

## ✨ Features

* ✅ Responsive Landing Page (Mobile + Desktop)
* ✅ Reusable Components (Navbar, Hero, Features, Footer)
* ✅ Smooth Navigation (Scroll to sections)
* ✅ API Integration (Mock API using JSONPlaceholder)
* ✅ Lead Capture Form (Frontend)

---

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/Dev23504/accredian-enterprise.git
```

2. Navigate to the project folder:

```
cd accredian-enterprise
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
